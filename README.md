# Currency-API
API Allows to get info about currencies, check rates of currencies

Example:

`http://localhost:4300/currencies/usd` returns 
```
{
    "USD": {
        "ID": "R01235",
        "NumCode": "840",
        "CharCode": "USD",
        "Nominal": 1,
        "Name": "Доллар США",
        "Value": 62.0315,
        "Previous": 61.7676
    }
}
```
